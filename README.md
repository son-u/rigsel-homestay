# Rigsel Homestay

This is the official website for **Rigsel Homestay**, located in Kaffer Gaon, Kalimpong, West Bengal.

## About the Project
* **Built with:** [Next.js](https://nextjs.org/)
* **Hosted on:** [Vercel](https://vercel.com/)
* **Live Site:** [rigselhomestay.in](https://rigselhomestay.in)

---

## Local Development

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
